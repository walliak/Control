#Control

电子设计3前2周课程内容安排：
一、熟悉开发版。【时间段3.7--3.14】
    设计自检程序。（显示数字，中文和图形（如直线、斜线、正弦波等），可以按键控制显示的内容）。

二、定时闹钟【时间段3.14--3.21】
设计要求：
1、能显示　时时－分分－秒秒。
2、能够设定定时时间、修改定时时间。
3、定时时间到能发出报警声或者点亮LED灯，或者启动继电器，从而控制电器的启停。
4、要求用中断延时而不是软件延时实现。

三、设计一个并行接口，分别实现软件延时和中断延时控制发光二极管。【时间段3.21--3.28】
1、控制发光二极管闪亮(使发光二极管每隔一秒改变一次状态)。
2、让二极管依次循环发亮,每次亮一个(间隔一秒,共16个)。
3、两个二极管发亮,各向相反的方向循环移动,依次发亮。
4、两个二极管发亮,各向相反的方向循环移动,依次发亮,移动速度不同。
5、用按键或拨码开关控制上述方式的转换。
6、比较两种延时方式的区别。

四、电机实验【时间段3.28--4.18】
1、认识步进电机和直流电机，查阅资料了解直流电机、步进电机的基本工作原理；学习用万用电表判断电机的绕组，并控制电机的正转和反转。
2、熟悉直流电机、步进电机的驱动电路的原理及设计；学习电机驱动模块（L298）的使用方法。
3、熟悉单片机控制直流电机、步进电机的应用：主要包括液晶屏显示转速或圈数/秒，用按键能设定和改变转速或圈数/秒。用按键控制电机正转或反转。（以不同速度正转、反转；按一下按键转一圈；按一下按键转1/4圈等）。
4、熟悉相关传感器及其应用：1、金属传感器与应用电路（集成金属传感器包括两种类型：电感式接近开关、电容式接近开关）；2、光电传感器、光敏二极管、光敏三极管（又分为可见光敏感型、红外光敏感型）；3、超声波传感器；4、霍尔传感器。
